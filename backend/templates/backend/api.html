API name Register
Explanation Send the device's information to server for registering
URL https://iot.ee.ncku.edu.tw/requesttoken/
Method POST
Parameter required Machine ID: machine mac address
User Information: user id
Response JSON format
{
"return_code":"200" ,
"token":token string
}
200 means OK.
